# Sync Portfolio

Portfólio pessoal do Sync, desenvolvedor focado em networking digital, criação de sites, APIs e automação.

## Sobre o projeto

Site desenvolvido com HTML, CSS e JavaScript puro. Design dark moderno com efeitos visuais sutis e performance otimizada.

## Tecnologias

- HTML5
- CSS3 (variáveis CSS, flexbox, grid)
- JavaScript vanilla
- Canvas API (partículas animadas)
- Google Fonts (Inter, Space Grotesk)

## Estrutura

```
├── index.html
├── style.css
├── script.js
└── README.md
```

## Características técnicas

- Design responsivo (mobile-first)
- Partículas animadas no background
- Glassmorphism nos cards
- Animações com intersection observer
- Cursor customizado com glow effect
- Parallax no hero
- Performance otimizada (sem frameworks)

## Paleta de cores
